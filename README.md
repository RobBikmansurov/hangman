# hangman

[Статья в Википедии](https://ru.wikipedia.org/wiki/Виселица_(игра))
	
## Technologies
Ruby

## Setup
To run this project, install the Ruby, save project locally using git and run it:

```bash
$ git clone git@github.com:RobBikmansurov/hangman.git
$ cd ./hangman
$ ruby main.rb
```
